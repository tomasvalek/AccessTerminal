Access Terminal
=======

Very simple Access Terminal as finite state machine in VHDL.

Access Codes:
-------
2049480717     
2049482512

Input signals:
-------
K : KEY     
CO : CNT_OF

Output signals:
-------
Mealy outputs:
FSM\_LCD\_WR, FSM\_LCD\_CLR

Moore outputs:
FSM\_CNT\_CE, FSM\_MX\_MEM, FSM\_MX\_LCD
